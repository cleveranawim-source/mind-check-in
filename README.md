# Teacher MindCare Check

대한민국 교사를 위한 마음건강 자가 점검 프로토타입입니다.

## 핵심 구성

- 36문항으로 소진, 불안, 우울, 분노, 도덕적 상처, 사건 후유 반응을 점검합니다.
- 결과는 진단명이 아니라 현재 두드러지는 회복 과제를 보여주는 유형입니다. 주 유형과 보조 신호를 함께 제공합니다.
- 위기 신호 문항은 점수와 별도로 판단하며, 체크 시 즉시 109, 112/119, 응급실, 신뢰할 수 있는 사람에게 연결하도록 안내합니다.
- 결과 화면은 상태 해석, 오늘의 대처, 이번 주 회복 설계, 동료/관리자/전문가 대화 문장, 공식 도움 경로를 제공합니다.

## 실행

```bash
npm install
npm run dev
```

## 안전 원칙

이 앱은 심리검사, 진단, 치료를 대신하지 않습니다. 교원 마음건강 회복지원, 교원치유지원센터, 정신건강복지센터, 정신건강의학과 상담으로 이어가기 위한 자기 점검 도구입니다.
